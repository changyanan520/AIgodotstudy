Godot 环境配置（C# 版）与 AI 环境
======================================

Godot 是一款开源免费的游戏引擎，支持 2D 和 3D 开发。C# 作为脚本语言拥有强类型检查、丰富的 .NET 生态，适合中大型项目。

一、Godot 环境配置（C# 版）
----------------------------

版本要求
^^^^^^^^

.. list-table::
   :widths: 20 30 50
   :header-rows: 1

   * - 工具
     - 版本
     - 说明
   * - Godot
     - 4.6.2.stable.mono
     - 必须选 .NET 版本，标准版不支持 C#
   * - .NET SDK
     - 8.0.21
     - Godot 4.6 当前推荐版本

**Godot .NET 版和标准版的区别：**

标准版只支持 GDScript，.NET 版同时支持 GDScript 和 C#。下载时认准 **Godot Engine – .NET** 标识。

基本概念速览
^^^^^^^^^^^^

- **场景（Scene）**：游戏中的每一个物体都是场景，由节点树组成
- **节点（Node）**：场景中的最小功能单元，如 Sprite2D、Camera2D
- **脚本（Script）**：挂载到节点上的代码，控制节点行为
- **项目（Project）**：整个游戏的容器，包含所有场景和资源

环境搭建
^^^^^^^^

1. **安装 .NET SDK**：在浏览器打开 https://dotnet.microsoft.com/download/dotnet/8.0 ，点击 x64 下载安装。验证：Win+R → cmd → ``dotnet --version`` 显示 8.0.xxx 即成功

2. **下载 Godot**：在浏览器打开 https://godotengine.org/download ，点击 .NET 版本下载，解压到指定文件夹

3. **创建项目**：双击 Godot.exe → 新建项目 → 填写名称和路径（不要用中文）→ 创建并编辑

二、AI 环境配置
-----------------

     - 性价比较高

配置步骤
^^^^^^^^

**第一步：下载安装 OpenCode 桌面版**

**第二步：配置 AI 模型路径**

1. 询问小新获取 **Dobest AI Coding** 使用配置说明
2. 按说明配置 OpenCode 的 AI 路径
3. **建议**：在配置新模型时，修改显示名称标注价格等级，方便选择

**第三步：开始使用**

1. 打开 OpenCode 桌面版
2. 在左侧栏打开项目所在路径
3. 下方切换工作模式：

   - **Plan 模式**：只分析不改动代码，用于方案设计
   - **Build 模式**：实际编写和修改代码

4. 右侧选择对应的 AI 模型
